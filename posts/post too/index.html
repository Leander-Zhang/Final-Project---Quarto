In this pair of data visualisations, an affective design method: Metaphorical Imagery was applied.

The only difference in these two visualisation is that the affective version has the background image. In this background image, the actual dog leash was in line with the plot line, metaphorically representing the drop in injury and death rate if more dog owners are using dog leash.

Development Process

For the “normal” version of this data visualisation, I used R studio to generate the plot:

#png("dogleash.png", width = 600, height = 600, bg = "transparent")

par(mar=c(5,5,5,5), bg="white", pin= c(2.8,2))

xnum <- c(45.5, 51.3, 57.8, 62.4, 70, 76.9)
ynum <- c(7.3, 6.2, 6, 5.5, 4.9, 3.6)


plot(xnum, ynum, type = "o", col = "red", lwd = 2,
     xlab = "Percentage of dog owners using DOG LEASH", 
     ylab = "Annual rate of injury or death of puppies", 
     main = "Buy a DOG LEASH, \n Keep your puppy safe.",
     xlim=c(45,80), ylim = c(3, 7.5), yaxs = "i", yaxt = "n", xaxt="n",
     cex.lab=0.8)

axis(2, at = seq(3, 7.5, by = 1), labels = paste0(seq(3, 7.5, by = 1), "%"), las=2, cex.axis=0.8)

axis(1, at = seq(45,80, by = 5), labels = paste0(seq(45,80, by=5), "%"), cex.axis=0.8)


segments (42.5,7.3,45.5,7.3,lty = 2)
segments(45.5,7.3,45.5,3,lty = 2)
segments(76.9,3.6,76.9,3,lty = 2)
segments(42.5,3.6, 76.9, 3.6, lty = 2)

text(xnum[1], 2.9, labels = paste0(xnum[1], "%"), pos = 1, col = "blue", cex = 0.9,xpd=T)  
text(44, ynum[1], labels = paste0(ynum[1], "%"), pos = 2, col = "blue", cex = 0.9,xpd=T)  
text(xnum[length(xnum)], 2.9, labels = paste0(xnum[length(xnum)], "%"), pos = 1, col = "blue", cex = 0.9,xpd=T)  
text(44, ynum[length(ynum)], labels = paste0(ynum[length(ynum)], "%"), pos = 2, col = "blue", cex = 0.9,xpd=T) 
#dev.off()


For the affective version, I drew the background image and then used PhotoShop to combine them together.
